# Movie app



# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)


# Getting started
- Clone the repository
```
git clone  https://github.com/Angemichel12/Movie-app.git
```
- Install dependencies
```
cd Movei-app
npm install
```
- Build and run the project
```
npm start
```
  Navigate to `http://localhost:300`
